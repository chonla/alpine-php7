# chonla/alpine-php7
-----

Docker image create php7 and [some components](#components) on top of alpine linux.

## Components

* git
* curl
* openssl
* php7
* php7-curl
* php7-iconv
* php7-json
* php7-mcrypt
* php7-openssl
* php7-pdo
* php7-pdo_sqlite
* php7-phar
* sqlite
